# Django 6 Hello World

The following is an example of Hello World in Django 6.

## Requirements:
```
Django==6.0.2
Python>=3.11
```

## Run the following commands in sequence to deploy the project to a development environment:

```bash
$ pip install -r requirements.txt

$ cp django_6_hello_world/settings.py_example django_6_hello_world/settings.py

$ python manage.py runserver
```

## Test the project:

Open your browser to http://127.0.0.1:8000 and you'll see the hello message.
